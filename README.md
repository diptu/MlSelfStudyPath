## 🛠️ Tech Stack & Tools
- Languages: Python (Primary), JavaScript (for Web-ML integrations)

- Libraries: PyTorch, FastAPI, NumPy, Pandas, Scikit-learn

- Environment: Mamba / Conda for isolated dependency management

- Formatting: LaTeX for mathematical derivations in .md files

  ## 📈 Key Learning Objectives
- Scratch Implementations: Rebuilding algorithms (like Backprop or K-Means) using only NumPy before moving to PyTorch.

- Theoretical Rigor: Completing all official Stanford problem sets and deriving mathematical proofs for loss functions.

- Modern LLM Training: Mastering the transition from standard Transformers to efficient training/fine-tuning (CS336).
